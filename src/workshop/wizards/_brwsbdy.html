Introduction to the Report Wizard
This Report Wizard guides you through the following steps to create a database field report:

1. Define the appropriate query for the report. WebSpeed uses the query
to display selected fields associated with one or more database tables.
 
2. Customize the style of the report. For example, you can specify the number of rows,
add a background color or image, and include a search form or navigation panel.
 
3. Add links from the report to other Web Objects.
 
4. Save the report to a local file, and preview the result.









Query
Use the following Query Builder applet to define the appropriate query for the report.
WebSpeed uses this query to display selected fields associated with one or more databases.






Style
To customize the design and layout of the report, specify the following options.

Title:    Enter the title you want to display at the top of the report.
 
Search Form: Yes     No Specify whether to use a search form so the user can find specific database fields. If you select Yes, the search form uses the first database field you defined in the Sort By section of the Query Builder applet.
 
Navigation Panel: Yes     No Specify whether to use a Navigation Panel to scroll through the report results.
 
Column Headings: Yes     No Specify whether to use the database field labels as column headings in the report.
 
Rows: Specify the number of rows you want to display in the report.
 
Border Size: Specify the size of the border for the report table.
 
 
Page Background: Specify the color or image for the background of the report page.
 
    Image: Image file located in the document root directory.
 
 
Table Background: Specify the color or image for the background of the display table in the report.
 
    Image: Image file located in the document root directory.

Links (optional)
Specify the following information to add hyperlinks to other Web Objects.

HyperLink Field: Select the database field that you want to display as the link text.
 
HyperLinked Web Object:   Enter the filename of the Web Object to run when a user selects the hyperlink. (Select the Files button to select from a list of existing Web Objects.)
 
Target Frame: Enter the name of the frame in which to run the Linked Web Object. The default frame, _self, overlays the report frame.
 
Status Line: Enter the text to display in the Status Line when a user places the mouse cursor over the Link Field.
