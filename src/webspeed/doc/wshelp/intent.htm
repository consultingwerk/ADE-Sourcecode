Internal Entries and Parameters

The Internal Entries and Parameters page displays a list of internal procedures and functions in a remotely running SmartDataObject (SDO). It contains information similar to what you see in the Procedure Object Viewer PRO*Tool for a locally running SDO. This information includes:

Internal Entries
A list of the internal procedures and functions in the SDO.
Parameters
A list of input and output parameters, plus any return values associated with functions.
