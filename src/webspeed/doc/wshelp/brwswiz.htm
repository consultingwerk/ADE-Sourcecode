Report Wizard

The Report Wizard creates a tabular view of the database fields you specify. Optionally, you can add navigation buttons for displaying the first, previous, next, and last set of records. You can also include a text entry field that allows the user to enter search criteria. (The user interface of the wizard is self-explanatory, therefore a further description in this help file is not necessary.)

If you are creating a complex Web object, you might need to load the Wizard-generated Web object into Workshop and modify the SpeedScript for it to work optimally.

For example, if you are defining tables within the query as external, when you are ready to move your application into production, you might want to remove the "No Link Table. Find first record for stand-alone testing" message and replace it with the functionality appropriate for your application.
