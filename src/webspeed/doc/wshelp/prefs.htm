Preferences

The Preferences frame allows you to set certain properties for the workshop session. This frame contains the following interface elements:

Submit
Submits the changes you made to the form.

Note: When you submit changes, Workshop writes a disk file to your working directory called wsprefs.dat. This file is read the next time you start Workshop in the same directory.
 
Reset
Resets all fields to the values when the frame was displayed, or since the last Submit.
 
Qualify Fields with Database Name
Automatically inserts the database name for all table-field references.
Workshop Timeout Period
Sets the time-out period for the Workshop session.
