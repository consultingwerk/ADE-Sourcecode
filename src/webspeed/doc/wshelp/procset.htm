Procedure Settings

The Procedure Settings frame allows you to set certain properties for the procedure. It contains the following items:

Submit
Submits the changes you made to the form.
 
Reset
Resets all fields to the values when the frame was displayed, or since the last Submit.
 
Procedure Type
Displays the procedure type.
Compile on Save
Specifies whether to automatically compile the procedure when you save it. The default is Yes.
Custom Lists
Allows you to specify the names of custom lists. WebSpeed Workshop provides six preprocessor references to store the names of field lists. In your code you can reference the list of fields indirectly by using the preprocessor reference. The six preprocessor names are {&LIST-1}... {&LIST-6}.
