Report Wizard

WebSpeed wizards let you create Web objects without having to write any HTML or SpeedScript code. The wizards prompt you for the necessary information and automatically generate the required code.

The Report Wizard creates a tabular view of the database field you specify. Optionally, you can add navigation buttons for displaying the first, previous, next, and last set of records. You can also include a text entry field that allows the user to enter search criteria.

You can also add a link from any one of the displayed fields to another Web object.
