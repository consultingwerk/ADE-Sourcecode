Query Builder - Select Records

The Select Records page allows you to define search criteria (filter expressions) for your query. The Select Records page contains the following elements:

Tables
Displays the tables selected for the query, and allows you to select the table containing the field for which you want to define selection parameters.
 
Primary field Selected for External Table
Specifies that this Web Object will find the same record passed to it from a parent Web Object.
 
Edit Box
Allows you to type in the WHERE expression, or you can select the Expression Editor.
 
Note: If you type in the expression here, only the fields for the selected table and those tables appearing before the selected table in the list are available for the expression.
 
Expression Editor
Displays the Expression Builder for the selected table.
