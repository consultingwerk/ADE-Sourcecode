New Section Dialog Box

The New Section dialog box allows you to create a template for a new code section. This dialog box contains the following elements:

Procedure
Activates the Procedure/Function Name edit box where you enter a name for the procedure.
 
Note: To override an ADM event procedure, enter the name with the prefix "local". For example, enter "local-display fields to override "adm-display-fields" in an HTML-Mapping Web Object.
 
Function
Activates the Procedure/Function Name edit box and the Function Return Type selection list.
 
Function Return Type
Allows you select a data type for the function return value from the drop down list
 
Procedure/Function Name
Allows you to type a name for your new procedure or function.
 
Control Handler
Activates the Events drop down list.
 
OK
Opens a code template for your selection in the Editor and adds the section name to the code section list.
 
Cancel
Cancels any selection and removes the dialog box.
