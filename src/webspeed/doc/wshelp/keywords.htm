Keywords Tool

The Keywords tool allows you to search on-line SpeedScript documents using reference subjects and keywords. This tool contains the following elements:

Search WebSpeed Reference
Allows you to search under one or more of the following reference subjects:
  • SpeedScript
  • Event Procedures
  • Methods
  • Preprocessors
  • SQL Reference
  • System Administration
Enter Search String
Specifies a keyword for your search. Wildcards may be used.
Submit Query
Submits the query to display results.The search function looks for the keyword appearing in topic titles within the selected reference subjects.
 
Matches
Displays matching files under each subject selected.
