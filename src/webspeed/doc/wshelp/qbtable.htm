Query Builder - Select Tables

The Select Tables page allows you to select the database and tables for your query.

Note: This is the first page in the Report Wizard. For an overview of the wizard, select the Intro tab.

The Select Tables Page contains the following elements:

Database
Displays the current database selection. Allows you to select another database from the drop down menu.
Tables
Displays all of the tables in the selected database. Allows you to select a table or tables for your query:

> Copies selected items from the Tables list to the Tables in query selection list.

>> Copies all items from the Tables list to the Tables in Query selection list.

Tables in Query
Displays the tables currently selected for the query. Allows you to remove tables:
 
< Removes selected items from the Tables in Query.

<< Removes all items from the Tables in Query list.
Up
Moves a selected table up in the Tables in Query selection list.
 
Down
Moves a selected table down in the Tables in Query selection list.
The following buttons appear on each page of the Report wizard:
 
Back
Displays the previous page in the Query Builder.
 
Next
Displays the next page in the Query Builder.
