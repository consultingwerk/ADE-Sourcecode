Query Builder - Join Tables

The Join Tables page allows you to include fields from more than one table in your query. The Join Tables Page contains the following elements:

Join Expressions
Displays a join expression created using the Add or Modify buttons.
 
Add
Displays the Join dialog box, which allows you to create a join expression.
 
Modify
Displays the Join dialog box for the join statement selected in the Join Expressions display.
 
Delete
Deletes the join expression selected in the Join Expressions display.
