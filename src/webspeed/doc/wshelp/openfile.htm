Opened Files

The Opened File frame displays the status of procedure files that are currently open in the Webspeed Workshop.

Note: If you are modifying any file in the list of opened files outside of Workshop, make sure you close the file first. This is because opened files are kept in Workshop memory when you edit the file using other editors.

This frame contains the following elements:

File
Displays the name of the file.
Type
Displays the file type (extension).
Needs to be Saved
Displays Yes if the file has been modified but not saved.
