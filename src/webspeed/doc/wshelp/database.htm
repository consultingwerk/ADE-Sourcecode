Databases Tool

The Databases tool displays a list of the fields and tables in the databases currently connected to the WebSpeed Agent. These databases might be different from the ones the AppBuilder is connected to. This tool contains the following interface elements:

Connected Databases
Allows you to select a database to view.
 
Table Listing:
Displays a list of database tables.
 
Field Listing:
Displays a list of fields and indexes. Click the field or index name to view definitions.
