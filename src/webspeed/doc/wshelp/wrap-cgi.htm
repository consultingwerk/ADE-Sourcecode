CGI Wrapper Template

Use the CGI Wrapper template to modify a SpeedScript procedure (.p) so that it can run as a Web object generating all the HTML for your Web page. This template includes the basic syntax you need for creating a state-aware Web application (the output-header procedure and the process-web-request procedure). The template has areas for descriptive comments, variable definitions, and code blocks.
