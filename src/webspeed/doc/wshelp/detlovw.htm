Detail Wizard

WebSpeed wizards let you create Web objects without having to write any HTML or SpeedScript code. The wizards prompt you for the necessary information and automatically generate the required code.
 
The Detail Wizard creates an updateable form to display a single database record, or single row of a join result, based on the selection criteria you specify. Optionally, you can add transaction control buttons to allow users to add, submit, reset, and cancel their updates.
