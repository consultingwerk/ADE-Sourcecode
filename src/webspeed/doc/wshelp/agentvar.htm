Agent Variables Tool

The Agent Variables Web Tool allows you to list environment variables and their settings for the current WebSpeed session. This tool contains the following interface elements:

List Variables
Choose one or more of the following options:
  • CGI -- CGI environment variables
  • WebSpeed -- WebSpeed environment variables
  • Web-Context -- WebSpeed WEB-CONTEXT attributes
  • Session -- WebSpeed SESSION attributes
Get List
Displays or refreshes the list of environment variables for the current WebSpeed session.
