Query Builder - Sort Records

The Sort Records page allows you to specify the sort order for your query. You must specify a sort order if you plan to use a search form. This page contains the following elements:

Table
Displays the currently selected table to use in the sort. Allows you to select another table from the drop down list.
Fields
Displays the available fields for the selected table. Allows you to select the field or fields by which you want the records sorted:
 
> Copies selected items to.

>> Copies all fields in the table to the Sort Fields list.
 
Sort Fields
Displays the field or fields currently selected as the sort field. Allows you (by double-clicking) to toggle between ascending and descending order:
< Removes selected fields from the Sort Fields list.

<< Removes all fields from the Sort Fields list.
Up
Moves a selected field up in the Sort Fields selection list.
 
Note: To specify an external table that contains the primary record for your query, move the table to the top of the list.
 
Down
Moves a selected field down in the Sort fields selection list.
