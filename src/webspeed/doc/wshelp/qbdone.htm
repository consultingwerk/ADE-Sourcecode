Query Builder - Done

The Done page displays the SpeedScript generated for your query. This page contains the following elements:

Back
Allows you to go back to previous pages to edit items as needed.
 
Select one of the Report Wizard tabs at the top of the frame:
 
Style tab
Displays the Style page, which allows you to define styles such as the number of rows, background color, a search form and a navigation panel.
 
Links tab
Displays the Links page, which allows you to link the report to other Web Objects.
 
Save tab
Saves the result to a local file and allows you to preview it.
