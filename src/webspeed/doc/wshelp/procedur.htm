Procedure Template

Use the Procedure template to write a SpeedScript procedure (.p). This template includes the basic syntax you need for creating a procedure. The template has areas for descriptive comments, variable definitions, and code blocks.

Note: This template should be used for SpeedScript files that update databases or perform utility functions that do not use SpeedScript extensions for reading from or writing to the Web.
